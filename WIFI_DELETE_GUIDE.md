# WiFi 删除功能位置说明

## ✅ 删除功能已实现，共有 2 个入口

### 1️⃣ 单条删除（删除单个 WiFi）

**位置**: 已保存网络 Tab → 每个 WiFi 卡片的右侧

**UI 示例**:
```
┌───────────────────────────────────┐
│ 🔖 Home-WiFi              🗑️     │  ← 点击这个红色删除图标
│ 🔒 ••••••••                       │
└───────────────────────────────────┘
```

**操作步骤**:
1. 进入 "WiFi 配置" 页面
2. 切换到 "已保存网络" Tab
3. 找到要删除的 WiFi 卡片
4. 点击右侧的**红色删除图标** 🗑️
5. 在弹出的确认对话框中点击 "删除"

**确认对话框**:
```
┌─────────────────────────┐
│   删除 WiFi             │
├─────────────────────────┤
│                         │
│ 确定要删除 Home-WiFi     │
│ 的配置吗？              │
│                         │
├─────────────────────────┤
│   [取消]     [删除]     │
└─────────────────────────┘
```

## 🔍 如何知道删除成功？

### 单条删除成功
1. 显示绿色提示条：`"WiFi配置已删除"`
2. 该 WiFi 卡片从列表中消失
3. 顶部计数减少（如：从 "已保存 2 个" 变为 "已保存 1 个"）

### 清除所有成功
1. 显示橙色提示条：`"所有WiFi配置已清除"`
2. 页面自动返回设备详情页
3. 设备重启（约 5-10 秒）
4. 再次进入时，显示 "没有已保存的 WiFi"

## ❓ 常见问题

### Q: 为什么看不到删除按钮？

**可能原因**:
1. 列表为空（没有已保存的 WiFi）
2. 加载中（按钮被禁用）
3. 没有切换到 "已保存网络" Tab

**解决方案**:
1. 确认切换到了 "已保存网络" Tab（不是"可用网络"）
2. 等待加载完成
3. 确认设备上真的有已保存的 WiFi

### Q: 点击删除按钮没反应？

**可能原因**:
1. 正在加载中（按钮被禁用）
2. 蓝牙连接断开

**解决方案**:
1. 等待当前操作完成
2. 检查蓝牙连接状态
3. 重新进入页面

### Q: 删除后还能恢复吗？

**答案**: ❌ 不能！删除操作是**不可恢复**的。

删除后：
- 单条删除：该 WiFi 的配置信息永久删除
- 清除所有：所有 WiFi 配置永久删除，设备重启

如果误删，需要重新配置 WiFi。

### Q: "清除所有"和"单条删除"有什么区别？

| 功能 | 单条删除 | 清除所有 |
|------|---------|---------|
| 删除范围 | 单个 WiFi | 所有 WiFi |
| 断开连接 | 否 | 是 |
| 设备重启 | 否 | 是 |
| 确认提示 | 简单确认 | 严重警告 |
| 操作时间 | 立即完成 | 需等待重启 |

## 💡 使用建议

### 何时使用"单条删除"？
- ✅ 删除不再使用的 WiFi
- ✅ 删除密码错误的 WiFi
- ✅ 整理 WiFi 列表

### 何时使用"清除所有"？
- ✅ 设备要转给其他人
- ✅ 重置所有网络设置
- ✅ 排查网络问题时重新开始

### 安全提示
- ⚠️ 删除前确认不再需要该 WiFi
- ⚠️ 清除所有会导致设备断网
- ⚠️ 记住 WiFi 密码，删除后需要重新输入
